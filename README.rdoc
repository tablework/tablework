== Tablework
Tablework is the leading app in preparing for roles, developing skills for engaging performances and powerful presentations.


=== Getting Started
Run bin/setup to install gems and prepare the db

=== Dependencies
We use hstore internally to store variable fields.
You will need to install postgres-contrib if you are on Ubuntu. 
# sudo apt-get install postgres-contrib	

=== Tests
Run tests with rake spec

=== Ruby Version
Tablework runs on Ruby 2.1.3
